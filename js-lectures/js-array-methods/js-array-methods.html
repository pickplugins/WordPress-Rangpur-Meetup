JavaScript Array Methods

Method: .toString

The JavaScript method toString() converts an array to a string of (comma separated) array values.

var fruits = ["First Name", "Last Name"];
document.getElementById("toString").innerHTML = name.toString();

Method: .join

The join() method also joins all array elements into a string. It behaves just like toString(), but in addition you can specify the separator:

var fruits = ["First Name", "Last Name"];
document.getElementById("result-join").innerHTML = name.join(" * ");

Method: .pop

The pop() method removes the last element from an array, The pop() method returns the value that was "popped out":

var arrayName = ["First Name",  "Middle name", "Last Name"];
values = arrayName.pop();
document.getElementById("result-pop").innerHTML = arrayName.join(" - ");
document.getElementById("result-pop-return").innerHTML = values;

Result:

Method: .push

The push() method adds a new element to an array (at the end), The push() method returns the new array length:

var arrayName = ["First Name",  "Middle name"];

document.getElementById("result-push").innerHTML = arrayName.join(" - ");
document.getElementById("result-push-return").innerHTML = values;

new_values = arrayName.push('Last name');
document.getElementById("result-push-new").innerHTML = arrayName.join(" - ");

Method: .shift

Shifting is equivalent to popping, working on the first element instead of the last. The shift() method removes the first array element and "shifts" all other elements to a lower index.

var arrayName = ["First Name",  "Middle name", "Last name"];
document.getElementById("result-shift").innerHTML = arrayName.join(" - ");

new_values = arrayName.shift();
document.getElementById("result-shift-new").innerHTML = arrayName.join(" - ");

Result:

Method: .unshift

The unshift() method adds a new element to an array (at the beginning), and "unshifts" older elements:

var arrayName = ["First Name",  "Middle name", "Last name"];
document.getElementById("result-unshift").innerHTML = arrayName.join(" - ");

new_values = arrayName.shift();
document.getElementById("result-unshift-new").innerHTML = arrayName.join(" - ");

new_values = arrayName.unshift('First name');
document.getElementById("result-unshift-final").innerHTML = arrayName.join(" - ");

Method: .splice

The splice() method can be used to add new items to an array: The splice() method returns an array with the deleted items:

The first parameter (2) defines the position where new elements should be added (spliced in). The second parameter (0) defines how many elements should be removed. The rest of the parameters ("Lemon" , "Kiwi") define the new elements to be added. 

var arrayName = ["First Name",  "Middle name", "Last name"];
arrayName.splice(2, 0, "Age", "Gender");

document.getElementById("result-splice").innerHTML = arrayName.join(" - ");

Method: .concat

The concat() method creates a new array by merging (concatenating) existing arrays, The concat() method does not change the existing arrays. It always returns a new array. 

var arrayName = ["First Name",  "Middle name", "Last name"];
arrayName.splice(2, 0, "Age", "Gender");

document.getElementById("result-splice").innerHTML = arrayName.join(" - ");

Method: .slice

The slice() method slices out a piece of an array into a new array. The slice() method creates a new array. It does not remove any elements from the source array. 

var arrayName = ["First Name",  "Middle name", "Last name", "Age", "Gender"];
var userData = arrayName.slice(2);

document.getElementById("result-slice").innerHTML = userData.join(" - ");
